## Getting Started (3 Steps)

### Step 1: Install (One-time, ~5 minutes)
```powershell
# Install Podman
choco install podman -y

# Install podman-compose
pip install podman-compose

# Verify
podman --version
podman-compose --version
```

### Step 2: Initialize Machine (One-time, ~30 seconds)
```powershell
# Create and start Podman machine
podman machine init
podman machine start

# Verify it's running
podman machine list
```

### Step 3: Run Your Application

## Using Podman Daily

```powershell
# Start all services
podman-compose up -d

# View live logs
podman-compose logs -f

# Stop all services
podman-compose down

# Restart a service
podman-compose restart backend

# Execute command in container
podman-compose exec backend python -c "import os; print(os.environ.get('OPENAI_API_KEY', 'NOT SET'))"

# Get container shell
podman-compose exec backend /bin/bash

# View container status
podman-compose ps

# Monitor resource usage
podman stats
```
